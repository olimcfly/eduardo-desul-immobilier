# Beckhamm Immo - Theme WordPress

Theme WordPress de base, propre et conforme aux standards, cree a partir de la structure du site source:
- [https://www.beckhamm-immobilier-aix-en-provence.fr/](https://www.beckhamm-immobilier-aix-en-provence.fr/)

## Installation

1. Copier le dossier `beckhamm-immo` dans `wp-content/themes/`
2. Aller dans **Apparence > Themes**
3. Activer **Beckhamm Immo**
4. Configurer les menus:
   - `Primary Menu`
   - `Footer Menu`
5. Definir une page d'accueil dans **Reglages > Lecture**

## Fichiers inclus

- `style.css` (entete theme WordPress)
- `functions.php` (supports theme, menus, assets)
- `header.php`
- `footer.php`
- `front-page.php`
- `index.php`
- `page.php`
- `single.php`
- `archive.php`
- `template-parts/content-none.php`
- `assets/css/theme.css`
